## Project
MetTraff: Targeted Feature Expansion of Meteorological Data for Dynamic Traffic Flow Prediction and Optimization.

## Deskripsi
Mengembangkan sistem manajemen lalu lintas cerdas berbasis Machine Learning. Sistem ini mengintegrasikan dataset arus kendaraan dengan variabel meteorologi (cuaca) dan fitur temporal (waktu) untuk memprediksi volume kendaraan secara akurat. Hasil prediksi tersebut kemudian digunakan sebagai basis optimasi durasi sinyal lampu lalu lintas secara dinamis.
